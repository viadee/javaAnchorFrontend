MAvA: Model Analysis via Anchor

This application aims to retrieve the patterns that a machine learning model has learned and make them understandable to people.
In addition, there is the possibility of finding several useful "anchors" in an automatic process.

The Anchors tool is used for this purpose.

The data as well as the models must be made available via the ML platform H2O.

Used libraries:

javaAnchorFrontend   javaAnchorServer   javaAnchorAdapters   javaAnchorExplainer
